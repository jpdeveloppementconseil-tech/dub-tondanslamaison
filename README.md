# JP Développement Conseil – Site Vitrine React

Site vitrine pour **JP Développement Conseil – by Du Béton dans la Maison**
Style **industriel chic**, responsive et moderne, avec un **formulaire de contact fonctionnel** via **Formspree**.

## Installation rapide

1. Installer les dépendances :
```
npm install
```

2. Démarrer en local :
```
npm start
```

3. Le site tourne sur http://localhost:3000

## Personnalisation
- Remplace `public/logo.png` par ton logo.
- Place tes images dans `public/images/`.
- Le formulaire envoie vers Formspree : https://formspree.io/f/mzzjobao (déjà configuré dans `ContactForm.js`).
